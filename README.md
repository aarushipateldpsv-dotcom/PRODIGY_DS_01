# Task 01: Data Visualization (Bar Chart / Histogram)

## 📌 Objective
Create a bar chart or histogram to visualize the distribution of a categorical or continuous variable such as age, gender, etc.

## 📂 Dataset
Sample dataset:  
https://github.com/Prodigy-InfoTech/data-science-datasets/tree/main/Task%201

## 🛠️ Tools & Libraries
- Python
- Pandas
- Matplotlib / Seaborn

## 🔍 Steps Performed
1. Loaded dataset using Pandas
2. Cleaned missing/null values
3. Selected relevant column (e.g., Age/Gender)
4. Created:
   - Bar chart (for categorical data)
   - Histogram (for continuous data)
5. Customized labels, titles, and colors

## 📊 Output
- Clear visualization of data distribution
- Insights into population characteristics

## 📌 Conclusion
The visualization helps in understanding patterns like most frequent categories or distribution trends.
